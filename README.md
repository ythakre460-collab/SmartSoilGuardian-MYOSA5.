---

## Overview

SmartSoil Guardian is an IoT-based smart agriculture system developed using ESP32. The system monitors soil moisture in real time and automatically controls irrigation.

It also performs predictive weed-risk monitoring using environmental conditions and sensor data.

### Key Features
- Automatic irrigation
- Soil moisture monitoring
- Water conservation
- Real-time monitoring
- Predictive weed-risk analysis

## Features (Detailed)

### Automatic Irrigation
The water pump automatically turns ON when soil moisture becomes low and turns OFF after sufficient moisture is achieved.

### Real-Time Monitoring
Sensor readings can be monitored continuously through the serial monitor and IoT platform.

### Weed-Risk Monitoring
The system analyzes environmental conditions for predictive weed-risk monitoring.

---

## Usage Instructions

1. Connect the ESP32 and sensors.
2. Upload the code to ESP32.
3. Power the system.
4. Monitor readings through serial monitor.

---

## Tech Stack

- ESP32
- Embedded C++
- Arduino IDE
- Soil Moisture Sensor
- Relay Module
- Mini Water Pump

---

## Requirements / Installation

```bash
Install Arduino IDE
Install ESP32 board package
Upload code using USB cable
```
